# Redesign Documentation Index

| File | Purpose |
| --- | --- |
| `v2-ui-redesign-roadmap.md` | Phase-by-phase execution plan for the v2 conversational UI, backend, and database upgrades. |

3. Update this README and both documents as milestones are completed.
